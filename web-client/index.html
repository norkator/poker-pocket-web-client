Readme
Poker Pocket is beta level under development project which goal is to become super simple and just play without messing around poker game.

Poker Pocket will never include content like half naked women or other content similar like that. It also wont pressure you to make you spent real money on coins whats so ever.

Slow progress comes from facts that I do this alone, except table and card graphics.

Poker Pocket is available for Android which has more game modes and Online Hold'em web browser version which is this. Both android and web browser online Hold'em games communicates with same server so having own account and playing with friends is possible.

Shuffling cards is based on cryptography level "secure random" algorithm.

Web version does not have ads since most of the people use ad blocker's. If you want to support, play more using Android app.
